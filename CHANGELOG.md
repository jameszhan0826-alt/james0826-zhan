# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-03-17
### Added
- Added "Regenerate Practice" functionality for "Brain Growing" and "Power Up Needed" words to ensure repeated training.
- Daily notifications now include specific words that need practice.

## [1.1.0] - 2026-03-17
### Added
- Improved practice sessions: Now generates both multiple-choice and fill-in-the-blank questions for every word.
- Added "Memory Tips" (mnemonics) to help users remember words better.
- Display memory tips during practice sessions.

## [1.0.0] - 2026-03-17
### Added
- Implemented "Review Test" feature in the Mastered tab, allowing users to practice words in a "Meaning -> Word" format.
- Added automatic word mastery tracking: words are marked as mastered after 3 correct answers.
- Added a changelog to keep users updated on new features and improvements.
